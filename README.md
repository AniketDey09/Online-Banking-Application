# Online-Banking-Application
Tech: Spring Boot, Spring Security, Hibernate, MySQL, Docker, AWS, Maven
Accomplished development of a secure banking system handling 1000+ simulated transactions by designing REST
API architecture and implementing microservices-based backend using Spring Boot.
Improved application security by 60% and achieved 99% uptime by implementing JWT authentication and
deploying containerized services using Docker and AWS.
Reduced API response time by 30% by optimizing SQL queries, indexing, and database performance in MySQL.
